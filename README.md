# repo-setup-sample

이 프로젝트는 Github 에서 Repository 를 사용 할 떄 관리해야 할 것들을 연습하기 위한 repository 입니다.

[See Demo](https://github.com/devSojoong/repo-setpup-sample)

## Prerequisite

- node.js : ^11.0
- java : ^8

## How to Start 
``` shell
$ git clone {repo-addr}
$ cd repo-setup-sample
$ npm install
$ npm start
```

## Features

- Blog
- Reply
- My page
- Donate with bitcoin
